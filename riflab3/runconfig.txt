To make a run configureation:
- create new OSGi framework run configuration
- deselect all bundles
- select riflab3, and set "Start Level" to 6 (Default Start Level: 4)
- select riflab3.entities
- select riflab3.logic1
- select riflab3.logic2
- select hu.bme.mit.mdsd.transportation.osgiLogReader
- select org.eclipse.equinox.log, and set "Start Level" to 5
- select org.eclipse.equinox.ds
- Add required bundles
- Settings: Clear the configuration area before lunching
